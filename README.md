# Phonetroller

Temporary Android placeholder app for a phone-based game/controller layout.

## What it does right now

- Forces landscape mode so the phone is held like a controller.
- Shows a polished controller UI with joysticks, sliders, and buttons.
- Lets you switch between controller profiles.
- Lets you add buttons, joysticks, and sliders visually.
- Lets you drag controls around while edit mode is on.
- Saves the controller layouts on the device.

This is only a visual placeholder for now. It does **not** connect to a PC/game yet.

## Download APK

Go to **Actions → Build APK → latest successful run → Artifacts → Phonetroller-debug-apk**.

The artifact contains `app-debug.apk`, which you can install on Android after allowing installs from unknown apps.
